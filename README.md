# 💄 My Personal Shade: HD Pro Edition

> **"하늘 아래 같은 핑크는 없다"** > 미세한 톤의 차이를 구분해내는 당신만의 감각을 증명하세요.

My Personal Shade는 단순한 게임을 넘어, 하이엔드 뷰티 브랜드의 감성을 담은 **디지털 컬러 조색 시뮬레이션**입니다. 섬세한 쉐이드 분석을 통해 당신의 절대 색감을 깨워보세요.

---

## ✨ Key Features

* **Professional Color Blending**: 가산 혼합 논리를 기반으로 한 정교한 조색 시스템.
* **Luxury UI/UX**: 영롱한 보석 테마의 타이틀과 유리 질감의 인터페이스.
* **Progressive Difficulty**: 스테이지가 거듭될수록 확장되는 컬러 스펙트럼과 프로 아티스트 레벨의 난이도.
* **Emotional Experience**: 은은하게 흐르는 하이엔드 스파 감성의 BGM과 핑크빛 벚꽃 엔딩 연출.

---

## 📖 How to Play

1. **Analyze**: 상단의 **TARGET 쉐이드**를 섬세하게 분석하여 필요한 색상을 유추합니다.
2. **Blend**: 하단의 컬러 칩을 캔버스로 **부드럽게 드래그**하여 혼합합니다.
3. **Complete**: 정교한 비율로 목표한 **시그니처 컬러**를 완성하세요.
4. **Master**: 새로운 챕터가 전개될수록 요구되는 **미세한 감각**에 도전하세요.
